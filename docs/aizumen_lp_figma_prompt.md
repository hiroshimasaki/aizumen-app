# AiZumen LP デザイン — Figmaプロンプト生成資料

---

## 1. プロダクト概要

**AiZumen（AI図面）** は、製造業（金属加工・板金・機械加工業など）向けの **見積・受注管理 SaaS** です。

| 項目 | 内容 |
|------|------|
| **ターゲット** | 中小製造業（町工場～中規模工場） |
| **コア価値** | AI OCR による注文書・図面の自動読取 → 見積自動作成 |
| **技術基盤** | React + Node.js + Supabase + Stripe + Google Gemini API |
| **料金体系** | 月額サブスクリプション（¥10,000～¥50,000）+ AIクレジット従量課金 |

### 主要機能一覧

| カテゴリ | 機能 |
|----------|------|
| **AI系** | OCR自動読取・一括解析・スマート見積・テナント別プロンプト |
| **見積管理** | 見積登録/編集/コピー・ステータス管理・重複チェック・印刷 |
| **スケジュール** | ガントチャート・納期管理・着手予定日・納品一括処理 |
| **分析** | ダッシュボード・収益分析・見積vs実績の差異分析 |
| **管理** | マルチテナント・ユーザー管理・権限制御・会社マスタ |
| **決済** | Stripe連携・プラン変更・AIクレジット追加購入・請求履歴 |

### 料金プラン

| プラン | 月額 | ユーザー数 | AI解析回数/月 |
|--------|------|-----------|-------------|
| **7日間 無料体験** | ¥0 | 1名 | 10回 |
| **Lite** | ¥10,000 | 2名 | 100回 |
| **Standard** | ¥30,000 | 10名 | 500回 |
| **Pro** | ¥50,000 | 50名 | 1,000回 |

## 2. SWOT分析

### Strengths（強み）

| # | 強み | 詳細 |
|---|------|------|
| S1 | **現場発・実運用ベース** | 正木鉄工で実際に使用中のツールが原型。机上の空論ではなく、製造現場の実需から生まれた |
| S2 | **AI OCR 自動読取** | Google Gemini APIによる注文書・図面のOCR解析で手入力を大幅削減 |
| S3 | **一気通貫の業務カバー** | 見積→受注→スケジュール→実績→分析まで1ツールで完結 |
| S4 | **ガントチャート内蔵** | 製造業の納期管理に特化した可視化ツールを標準搭載 |
| S5 | **低価格帯** | ¥10,000/月～で中小製造業でも導入しやすい |
| S6 | **マルチテナント・RLS** | PostgreSQL RLSによるデータ完全分離でセキュリティ確保 |
| S7 | **従量課金の柔軟性** | AI機能はクレジット制で使った分だけ。無駄なコストなし |

### Weaknesses（弱み）

| # | 弱み | 詳細 |
|---|------|------|
| W1 | **ブランド認知ゼロ** | 新規SaaSであり市場での知名度がない |
| W2 | **導入実績が限定的** | 正木鉄工1社のみの利用実績（ローンチ前） |
| W3 | **モバイル最適化が未成熟** | デスクトップ優先設計。スマホでの操作性は限定的 |
| W4 | **会計・ERPとの連携未実装** | 弥生・freee等との直接連携は今後の課題 |

### Opportunities（機会）

| # | 機会 | 詳細 |
|---|------|------|
| O1 | **中小製造業のDX遅れ** | Excel・紙ベースの見積管理が主流。デジタル化ニーズが急増 |
| O2 | **生成AI・OCR技術の進化** | Gemini等のマルチモーダルAIが図面解析精度を飛躍的に向上させている |
| O3 | **2025年の電子帳簿保存法対応** | 電子保存義務化で電子管理ツールの需要がさらに高まる |
| O4 | **補助金・IT導入補助金** | 中小企業向けIT導入補助金との親和性が高い |
| O5 | **製造業SaaS市場の成長** | 国内製造業SaaS市場は年20%以上で成長中 |

### Threats（脅威）

| # | 脅威 | 詳細 |
|---|------|------|
| T1 | **大手ERPベンダーの進出** | SAP・Oracle等が中小向け製品を拡充中 |
| T2 | **類似SaaSの台頭** | TerminalQ、Atsumori等の見積特化SaaSが競合 |
| T3 | **町工場の価格感度** | 月額3万円超は中小製造業にとって導入障壁になりうる |
| T4 | **AI精度への懐疑** | 図面のAI読取精度に対する現場の不信感 |

## 3. 競合分析

### 主要競合マッピング

| サービス | 特徴 | 月額目安 | AI/OCR | 弱点 |
|----------|------|---------|--------|------|
| **TerminalQ** | 製造業見積作成に特化 | ¥30,000～ | なし | AI解析なし・スケジュール管理なし |
| **Atsumori** | 見積書作成＋顧客管理 | ¥10,000～ | なし | 製造業特化ではない。汎用見積ツール |
| **Kintone** | 汎用業務アプリ基盤 | ¥780/ユーザー | プラグイン | カスタマイズに専門知識必要。重い |
| **Board** | 製造業向け受発注管理 | 要問合 | 部分的 | 高額。大手向け。中小には過剰 |
| **Excel/紙** | 現行のやり方 | ¥0 | なし | 属人化・検索不可・集計が困難 |

### AiZumenの競争優位性

```
                    AI解析あり
                        ▲
                        │
         AiZumen ●──────┤
                        │
    製造業特化 ◄─────────┼──────────► 汎用ツール
                        │
                  Board ●     ● Kintone
                        │
                        │    ● Atsumori
               TerminalQ ●
                        │
                        ▼
                    AI解析なし
```

**AiZumenのポジション**: 「製造業特化 × AI搭載」は現時点で **唯一のポジション**。

## 5. Figma プロンプト（コピー＆ペースト用）

以下のプロンプトをFigmaのAIデザイン機能にそのまま貼り付けてください。

---

### プロンプト本文

```
製造業向けSaaS「AiZumen（AI図面）」のランディングページをデザインしてください。

■ プロダクト概要
・製造業（町工場・金属加工・板金加工）向けの見積・受注管理クラウドサービス
・最大の差別化ポイント：AI OCR（Google Gemini API）で注文書・図面を自動読取し、品名・数量・加工費を自動入力
・見積登録 → 受注管理 → ガントチャート（スケジュール管理）→ 実績入力 → 収益分析まで一気通貫で対応
・実際の製造現場（正木鉄工）で運用実績あり
・月額¥10,000から利用可能。7日間の無料トライアル付き

■ ターゲットユーザー
・中小製造業の経営者・工場長・管理部門
・現在Excelや紙で見積書を管理している企業
・年代は40〜60代が中心、ITリテラシーは標準レベル
・課題：手入力に時間がかかる／見積が属人化している／納期管理が煩雑

■ デザインコンセプト
・カラー：ダークモードベース。背景は深いネイビー（#0B1120）。ブルー → シアンのグラデーションをアクセントに使用
・スタイル：グラスモーフィズム（半透明カード＋ブラー背景）。柔らかいグロウエフェクトの光球を背景に配置
・トーン：先進的だが信頼感がある。「町工場のDXを加速する」イメージ
・角丸は大きめ（16px〜24px）。シャドウは深めでリッチに
・文字は白系で読みやすく。日本語メイン

■ LP構成（セクション順）
1. ヒーローセクション
   - キャッチコピー例：「図面を読み込む、見積が生まれる。」または「AIが図面を読む。見積が変わる。」
   - サブコピー：「注文書をアップロードするだけ。AIが品名・数量・加工費を自動で読み取り、見積を即座に作成します。」
   - CTAボタン2つ：「無料で始める」（プライマリ、ブルー→シアングラデーション）、「機能を見る」（セカンダリ、ゴースト）
   - 背景に製品のダッシュボードUIのスクリーンショット（モックアップ）をフロートで配置
   - 大きなグロウ球エフェクト（ブルーとシアン）を背景に

2. 導入実績・信頼バー
   - 「製造現場で実証済み」のメッセージ
   - ロゴ群（例示用プレースホルダー or 「正木鉄工で運用中」）
   - 数字実績：「見積作成時間 80%削減」「手入力ミス ゼロへ」「導入2分で完了」

3. 課題提起セクション
   - 「こんなお悩みありませんか？」
   - 3〜4個のペインポイントをアイコン付きカードで表示
   - 例：「見積書をExcelで手入力している」「図面の内容を何度も転記している」「納期を見落としてしまう」「見積が特定の人しかできない」

4. 解決策セクション（メイン機能紹介）
   - 「AiZumenが解決します」
   - 3つのメイン機能をビジュアル付きで紹介：
     a) AI図面読取：「注文書PDFをドラッグ＆ドロップ → AIが自動で品名・数量・単価を解析」
     b) 一気通貫の案件管理：「見積 → 受注 → スケジュール → 実績 → 分析を1画面で」
     c) ガントチャート：「納期を一目で把握。遅延リスクを早期発見」
   - 各機能にUIスクリーンショット（モックアップ）を配置

5. 機能一覧セクション
   - グリッドレイアウトで6〜8個の機能カードを並べる
   - 各カードにアイコン＋タイトル＋1行説明
   - 例：「AI一括解析」「スマート見積」「重複チェック」「見積印刷」「ユーザー管理」「収益分析」「データ分析」「クレジット管理」

6. 料金プランセクション
   - 4列のプランカード：無料体験 / Lite / Standard / Pro
   - Standardに「人気」バッジ
   - 各カードに月額・ユーザー数・AI回数・含まれる機能を表示
   - 「全プランに+10クレジットボーナス」のバナー
   - 下にCTA「無料で体験する」

7. セキュリティ・信頼セクション
   - 「安心のセキュリティ」
   - 3〜4個のバッジ：SSL暗号化、データ完全分離（RLS）、自動バックアップ、Stripe決済
   - 小さなアイコン＋テキストのミニマル表示

8. FAQ セクション
   - アコーディオン形式で5問程度
   - 例：「無料体験に制限はありますか？」「データ移行はできますか？」「途中でプラン変更できますか？」

9. CTA セクション（最終）
   - 「今すぐ、見積業務をアップグレード」
   - 大きなCTAボタン「7日間無料で始める」
   - 「クレジットカード不要」の補足テキスト

10. フッター
    - ロゴ、ナビリンク（機能・料金・お問い合わせ）
    - 法的リンク（利用規約・プライバシーポリシー・特定商取引法）
    - コピーライト

■ デバイス対応
・デスクトップ（1440px）をメインでデザイン
・モバイル（375px）のレスポンシブも作成

■ その他の指示
・日本語テキストを使用
・製造業の信頼感（堅実さ）とAIの先進性のバランスを取る
・CTAボタンにはマイクロインタラクション（ホバーで光るエフェクト）を想定したデザイン
・スクロールに連動したフェードインアニメーションを前提としたレイアウト間隔
```
