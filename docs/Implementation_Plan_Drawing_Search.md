# 図面類似箇所検索機能 実装計画書

## 1. プロジェクト概要
スキャン済みPDF図面から、ユーザーが選択した特定の矩形領域（記号、部品、配管形状等）をクエリとして、蓄積された図面データベースから類似箇所を検索・サジェストする機能を実装する。
FAXやスキャンによるノイズ、図面の切り抜き、回転等の実運用上の課題に対応しつつ、APIコストを最小化する設計とする。

---

## 2. システムアーキテクチャ (ハイブリッド・リランキング方式)

| フェーズ | 処理内容 | 実行場所 | コスト | 使用技術 |
| :--- | :--- | :--- | :--- | :--- |
| **1. 登録 (Indexing)** | PDF画像化、タイル分割、ノイズ除去、ベクトル化 | ローカルサーバー | **無料** | pdf2image, OpenCV, MobileNet/SIFT |
| **2. 一次検索 (Search)** | ベクトルDBによる高速な粗絞り込み (上位30〜50件) | ローカルサーバー | **無料** | Faiss, Milvus, pgvector等 |
| **3. 二次精査 (Reranking)** | Gemini APIによる高度な幾何学的比較と順位付け | Google Cloud | **有料 (低)** | Gemini 1.5 Flash |

---

## 3. 実装ステップ詳細

### 3.1. 図面登録（インデックス作成）
* **PDF変換:** 300dpi以上の解像度で画像化。
* **前処理:** `cv2.adaptiveThreshold` 等を用いてFAXノイズ（背景のグレーや黒点）を除去。
* **タイル分割:** 図面をスライディングウィンドウで分割（例: 300x300px）。
* **ローカル特徴量抽出:** Gemini APIを使わず、オープンソースモデル（MobileNetV3, CLIP等）でベクトル化。
* **DB保存:** 特徴量ベクトルをFaiss等の近傍探索ライブラリに保存。

### 3.2. 検索実行
* **クエリ抽出:** ユーザーが画面上で選択した矩形領域を画像として切り出し。
* **一次フィルタ:** ローカルDBから「見た目の構成」が近いタイルを数十件抽出。
* **リランキング:** 候補をバッチ化し、Gemini APIへ送信。

### 3.3. Gemini APIによる精密判定
* **モデル:** `gemini-1.5-flash`
* **アプローチ:** 1枚のクエリ画像 vs 10枚の候補画像を1回のリクエストで比較（バッチプロンプティング）。
* **判定基準:** 幾何学的構造（角度、接続、配置）を重視し、ノイズを無視するよう指示。

---

## 4. 精度最大化プロンプト (Gemini用)

```markdown
システムプロンプト:
あなたは熟練の図面検図エキスパートです。
入力された「画像A（クエリ）」と、10枚の「候補画像（画像1〜10）」を幾何学的に比較してください。

判定ルール:
1. FAX特有のノイズ（黒点、掠れ）や線の太さは無視し、図形の構造のみを見ること。
2. 線の交差角度、円と線の接点、シンボル内のパーツ配置を最優先すること。
3. 90度/180度の回転があっても、構造が同じなら同一とみなすこと。
4. Step-by-Stepで各要素を分析した上でスコアを出すこと。

出力形式 (JSON):
[
  { "id": 1, "score": 95, "reason": "中心の円と斜線の構成が完全一致。ノイズを除けば同一記号と判断。" },
  ...
]
```

## 5. コスト試算 (想定)
* **構成:** 1クエリにつき候補10件をGemini 1.5 Flashで比較。
* **消費トークン:** 約3,500〜4,000トークン / 回。
* **概算単価:** 約 **0.12円 / 1検索**。
* **運用案:** 登録（無料）と検索（有料）を分離し、ユーザーが「検索」を押した際のみクレジットを消費させる。

---

## 6. Antigravity への指示案
以下のモジュール開発を依頼してください：
1. **Image Processor:** `pdf2image` と `OpenCV` を用いた、図面クリーニングおよびタイル分割機能。
2. **Local Vector Engine:** `MobileNetV3` 等を用いた、APIコストのかからない特徴量抽出・保存・検索機能。
3. **Gemini Reranker:** `google-generativeai` SDKを用いた、マルチモーダル画像比較およびリランキング機能。
