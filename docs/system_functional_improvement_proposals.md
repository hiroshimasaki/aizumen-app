# システム機能改善・追加提案レポート

現在の AiZumen の仕様に基づき、UI/UX 以外の機能面（ロジック、データ活用、業務効率化）を中心とした改善・追加の余地を提案します。

## 1. AI(OCR) 解析機能の高度化

### 1.1 学習型マッピング調整 (Few-shot Adjustment)
- **内容**: ユーザーが AI の解析結果（特にマッピング漏れ）を手動で修正した際、その修正パターンを特定テナント向けの「補足命令」として動的にプロンプトへ組み込む機能。
- **効果**: 使えば使うほど、その会社特有の書類フォーマットに対する解析精度が向上します。

### 1.2 信頼度スコアの返却とアラート
- **内容**: 各項目の抽出精度（Confidence Score）を AI から取得し、信頼度が低い項目（例: 70%未満）を UI 上で強調表示する。
- **効果**: 全項目を目視確認する手間を省き、エラーが発生しやすい箇所に集中して確認できます。

### 1.3 数量・金額の整合性チェック
- **内容**: 「単価 × 数量 = 合計金額」の計算が書類上の数値と一致するかを AI 側で検証し、不整合があれば通知する。
- **効果**: OCR の読み取りミス（「3」を「8」と誤認するなど）を論理的に検知できます。

## 2. 図面管理・検索機能の拡張

### 2.1 図面バージョン管理 (Revision Management)
- **内容**: 同じ図番で新しいファイルがアップロードされた際、上書きではなく履歴として保存し、過去の見積や発注書との紐付けを維持する。
- **効果**: 製作ミスに直結する「旧図面での加工」を防ぎます。

### 2.2 自動タグ付け (Auto-Tagging)
- **内容**: アップロードされた図面の内容から、材質（SS400, SUS304 等）、加工方法（旋盤、フライス、レーザー等）、表面処理の有無を AI が判定し、検索用メタデータとして自動付与する。
- **効果**: 詳細な検索条件（「SUSの旋盤加工品」等）での絞り込みが、事前のデータ入力なしで可能になります。

## 3. 案件・工程管理の高度化

### 3.1 ガントチャートの自動更新 (Auto-Sync)
- **内容**: 見積明細の状態（納品済等）が変更された際、関連するガントチャートの進捗率を自動で反映し、遅延が発生している場合に管理者へ通知する。
- **効果**: 進捗管理の二重入力の手間をなくし、デッドラインの見落としを防止します。

### 3.2 リソース負荷の可視化
- **内容**: 作業者（Worker）マスタを追加し、案件を割り当てることで、特定時期の稼働率を予測・可視化する。
- **効果**: 無理な納期回答の抑制や、外注検討の判断材料となります。

## 4. データ分析・経営支援 (BI)

### 4.1 異常検知アラート
- **内容**: 過去の類似案件の利益率と比較して、今回入力された見積の利益率が著しく低い場合に警告を表示する。
- **効果**: 入力ミスや、原価高騰による採算割れを防ぎます。

### 4.2 顧客別・材質別利益分析
- **内容**: 現在の売上集計だけでなく、どの顧客やどの材質の加工が高い利益を生んでいるかを分析するダッシュボード。
- **効果**: 営業戦略や設備投下判断の精度を高めます。

## 5. 外部連携・拡張性

### 5.1 Webhook による通知連携
- **内容**: 案件登録、受注、完了などのイベント時に Webhook を発行し、Slack, LINE, もしくは外部 ERP システムへデータを即時転送する。
- **効果**: 社内の既存ツールとの連携を容易にします。

### 5.2 大量ファイル監視エージェント (Hotfolder 強化)
- **内容**: PC の特定フォルダを常時監視し、図面ファイルを入れるだけで自動的に AI 解析・下書き登録まで完了させるデスクトップアプリの提供。
- **効果**: ブラウザへのアップロード操作すら不要にし、現場の入力負荷を極限まで下げます。

---
本提案は、AiZumen が「見積ツール」から「製造業のデジタル基盤」へと進化するためのロードマップとして検討可能です。
