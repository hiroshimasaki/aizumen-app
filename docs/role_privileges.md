# ユーザーロール別 権限一覧

AiZumen SaaS における各ユーザーロールの権限範囲を以下にまとめます。

## 主要ロールと役割

| ロール名 | 対象範囲 | 主な役割 |
| :--- | :--- | :--- |
| `super_admin` | プラットフォーム全体 | 全テナントの管理、システムログ参照、フォーラムの違反報告管理・お知らせ投稿。 |
| `system_admin` | 各テナント | 自テナントのユーザー管理、バックアップ実行、契約・設定の変更。 |
| `user` | 各テナント | 見積の作成・編集・削除、フォーラムへの投稿。 |
| `viewer` | 各テナント | 参照専用。見積内容の確認が可能（※操作制限は要強化）。 |

## バックアップ機能の改善提案（図面 PDF 一括ダウンロード）

現時点のバックアップ機能は、見積の JSON メタデータのみが対象であり、アップロードされた図面 PDF 本体は含まれていません。
一括ダウンロードを実装するための推奨アプローチは以下の通りです。

1.  **サーバーサイドでの ZIP 生成**:
    - `archiver` ライブラリをサーバーに導入し、`quotation-files` バケット内の全ファイルをストリームで ZIP 化して返すエンドポイントを実装。
    - メリット：1 クリック。
    - デメリット：ファイル数が多いとサーバー負荷。
2.  **クライアントサイドでの ZIP 生成**:
    - 全ファイルの署名付き URL を取得し、`jszip` や `client-zip` を用いてブラウザ上で ZIP 生成。
    - メリット：サーバー負荷なし。
    - デメリット：ブラウザのメモリ消費。
3.  **代替案 (推奨)**:
    - 毎日・毎週のバッチ処理で、Storage 内の全体ファイルを ZIP し、バックアップフォルダに保存しておく。
    - ユーザーは既に生成済みの ZIP をダウンロードするだけで済む。

今後、一括ダウンロード機能が必要な場合は、上記のうちバッチ処理による ZIP アーカイブ化の導入を検討することをお勧めします。
