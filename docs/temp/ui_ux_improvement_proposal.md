# UI/UX改善提案レポート : AiZumen

## 1. はじめに
本レポートでは、図面管理・見積支援プラットフォーム「AiZumen」のUI/UXに関する現状分析と、それに基づいた改善案を提案します。現在のプロダクトは既にダークテーマを基調としたモダンでプレミアムなデザインを採用していますが、さらなる利用体験（UX）の向上と視覚的洗練（UI）を目指します。

---

## 2. 現状分析
### 2.1 強み
- **プレミアムな質感**: ダークテーマ（#0c0e14）とグラデーション、背景のぼかし（backdrop-blur）の活用により、プロフェッショナルかつ高品質な印象を与えている。
- **一貫性**: Tailwind CSSとLucide Reactアイコンの活用により、コンポーネント間である程度の一貫性が保たれている。
- **高機能**: AI解析、ガントチャート、図面検索など、複雑な機能を直感的にレイアウトしようとする工夫が見られる。

### 2.2 課題点
- **情報密度の最適化**: 見積一覧やガントチャートにおいて、情報量が多い場合に視覚的な負荷が高くなる可能性がある。
- **インタラクションの深み**: 操作に対するフィードバック（マイクロインタラクション）がさらに洗練される余地がある。
- **モバイル体験**: 複雑な機能（特にガントチャートや一括解析）のモバイルでの操作性が改善できる。
- **空状態（Empty States）**: データがない場合の導線がシンプルすぎる。

---

## 3. 改善提案

### 3.1 視覚的洗練（UI）
#### A. デザインシステムの厳格化
- **カラーパレットの固定**: ブランドカラー（Indigo/Cyan）のHSL値を定義し、すべてのコンポーネント（ボーダー、影、背景）で再利用することで、より統一感のある「面」を作る。
- **タイポグラフィの階層化**: Noto Sans JPのウェイトをより細かく使い分け、数値や重要情報の視認性を高める。特に「見積金額」と「実績差異」のコントラストを強化する。

#### B. マイクロインタラクションの強化
- **状態遷移アニメーション**: ステータス変更時（例：検討中→受注）に、色がじわっと変わるようなトランジションや、パーティクル効果などの微細な演出を加える。
- **スケルトンローディング**: 現在のスピンナーに加え、コンテンツの形に合わせたスケルトン表示を採用し、体感待ち時間を短縮する。

---

### 3.2 ユーザー体験（UX）の最適化
#### A. ステップ型ウィザードの導入（一括解析）
- **一括解析フローの改善**: 現在のモーダルでの一括処理を、「アップロード」→「AI解析内容の確認（プレビュー付き）」→「一括登録」というステップ型UIに変更。ユーザーがAIの解析結果を確認・微修正してから登録できるフローにする。

#### B. インタラクティブなガントチャート
- **操作性の向上**: ガントチャート上でのドラッグ＆ドロップによる日程調整、ダブルクリックでの詳細編集、マウスホイールでのスムーズなズーム機能の実装。
- **警告表示**: 納期が迫っているタスクや、実績が予定を大幅に超えているタスクを、チャート上で点滅させたりアイコンで強調表示する。

#### C. コマンドパレット（Quick Action）の実装
- **パワーユーザー向け**: `Ctrl+K`などで、ページ内の検索、新規登録、ページ遷移をキーボードだけで完結できるコマンドパレットを導入。

---

### 3.3 コンポーネント別の具体的な改善案

| コンポーネント | 現状 | 改善案 |
| :--- | :--- | :--- |
| **案件一覧 (QuotationList)** | リスト形式のみ | カード形式とリスト形式の切り替え。状況（受注/失注）に応じたカード背景のダイナミックな変化の強調。 |
| **見積フォーム (QuotationForm)** | 長いスクロール形式 | セクションごとの「タブ分け」または「アコーディオン形式」を採用し、必要な情報に素早くアクセス可能にする。 |
| **図面検索モーダル** | 検索結果の一覧表示 | 検索元図面と検索結果を左右に並べた「比較ビュー」の実装。 |
| **ダッシュボード (Analysis)** | 表と数値がメイン | ヒートマップやトレンドグラフ（Rechartsなど）の活用により、経営状況を直感的に把握可能にする。 |

---

## 4. 実装へのアプローチ
1.  **CSS変数の整理**: `index.css`にデザインシステムに基づくカラー・サイズ変数を定義。
2.  **共通コンポーネントのブラッシュアップ**: `Button`, `Card`, `Input`といった最小単位のスタイリングを刷新。
3.  **機能別UX強化**: 一括解析フロー、ガントチャートの順で、高付加価値な体験改善を適用。

---

## 5. 結び
AiZumenは既に高いレベルのデザインを実現していますが、より「使う楽しさ」と「業務の確実性」を両立させるために、上記のような微細な調整とフローの最適化が有効です。これらにより、ユーザーのエンゲージメントと満足度をさらに高めることができると確信しています。
