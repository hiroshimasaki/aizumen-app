# マーケティング分析スキル (marketing-analysis)

## 目的
製造業・町工場向けSaaSのLPを、マーケティングおよび心理学的な観点から評価・改善するための指針を提供します。

## 分析の4本柱

### 1. 感情的コネクト (Emotional Connection)
- **現状肯定**: 「現場の大変さ」を正しく描写できているか？
- **経営者視点**: 経営者が直面している「属人化」「受注機会損失」という痛みに深く切り込めているか？
- **信頼感**: 開発者が「元製造業経営者」であるという強力なUSP（独自の売り）が前面に出ているか？

### 2. デザインの心理的効果 (Visual Psychology)
- **色彩設計**: 工業的な信頼感（ブルー、ネイビー）と、AI的な先進性（シアン、エメラルド）が調和しているか？
- **フォント**: `Roboto Mono` などの等幅フォントをアクセントに使い、製造業の正確性を演出できているか？
- **余白**: 高級感（プレミアム感）を出すための十分な余白が計算されているか？

### 3. メッセージングとコピーライト (Copywriting)
- **ベネフィット vs 機能**: 機能（AI解析）ではなく、ベネフィット（見積時間が1/10になる）が大きな文字で書かれているか？
- **マイクロコピー**: CTA（ボタン）のテキストが「無料で試す」だけでなく、心理的ハードルを下げる工夫（クレカ不要など）が含まれているか？

### 4. コンバージョン設計 (Conversion Optimization)
- **CTAの配置**: スクロール位置に応じて、迷わず次に進めるボタンが配置されているか？
- **社会的証明**: カスタマーボイスや想定ROIが具体的であるか？

## 実践手順
1. **競合調査**: 大手製造業DXツールとの視覚的差異を明確にする。
2. **コピーのA/B案作成**: ターゲットの属性に基づいた複数のヘッドライン案を提示する。
3. **継続的改善**: ヒートマップ（想定）に基づいた、要素の再配置を提案する。

## 推奨ツールスタック (2026年版)

### デモ動画・インタラクティブ体験
- **Screen Studio / FocuSee**: 操作画面の自動スーム・カーソル滑らか化。SaaSの「触り心地」を伝えるのに最適。
- **Supademo / Storylane**: ユーザーが自分で操作できるインタラクティブデモ。CVR向上に直結。
- **HeyGen / Synthesia**: AIアバターによる解説。多言語展開や親しみやすさの演出に。

### コピー・分析
- **Copy.ai / Jasper**: 製造業特有の用語を活かしたキャッチコピー生成。
- **Hotjar / Microsoft Clarity**: 実際のユーザー行動（ヒートマップ）の分析。
